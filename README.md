# 🛒 ILMIGROSIR – Katalog Harga Digital

Aplikasi katalog harga grosir berbasis web untuk **ILMIGROSIR** — *Jual Kembali Untung Berkali*.

## 🌐 Cara Hosting ke GitHub Pages

### Langkah 1: Buat Repository GitHub
1. Buka [github.com](https://github.com) dan login
2. Klik tombol **"New"** untuk buat repository baru
3. Beri nama repository, contoh: `ilmigrosir` atau `katalog-harga`
4. Pilih **Public**
5. Klik **"Create repository"**

### Langkah 2: Upload File
1. Di halaman repository, klik **"uploading an existing file"**
2. Upload file berikut:
   - `index.html` ← file utama aplikasi
3. Klik **"Commit changes"**

### Langkah 3: Aktifkan GitHub Pages
1. Buka tab **Settings** di repository
2. Scroll ke bawah, cari menu **"Pages"** di sidebar kiri
3. Di bagian **Source**, pilih branch **`main`** dan folder **`/ (root)`**
4. Klik **Save**
5. Tunggu 1–2 menit, lalu website aktif di:
   ```
   https://[username-github].github.io/[nama-repo]/
   ```
